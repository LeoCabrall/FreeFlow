# Functional Requirements (FR)

## Introdução

Este documento descreve todos os requisitos funcionais do FreeFlow.

Cada requisito representa uma funcionalidade que deverá ser implementada durante o desenvolvimento do sistema.

---

# Autenticação

## FR-001

O sistema deverá permitir cadastro de usuários.

## FR-002

O sistema deverá permitir login utilizando e-mail e senha.

## FR-003

O sistema deverá permitir recuperação de senha.

## FR-004

O sistema deverá permitir edição do perfil.

## FR-005

O sistema deverá permitir exclusão da conta.
