# FreeFlow

> Plataforma inteligente de gestão financeira desenvolvida com Java, Spring Boot, React e Oracle Database.

---

## Sobre o projeto

O FreeFlow é uma plataforma desenvolvida como projeto acadêmico da FIAP, com foco em gestão financeira, automação de processos e organização de informações financeiras.

O objetivo é fornecer uma solução moderna para controle financeiro pessoal, permitindo que usuários acompanhem receitas, despesas, metas, investimentos e indicadores financeiros em um único ambiente.
